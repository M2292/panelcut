Favicon Setup for PanelSnip
=============================

Current favicon: favicon.svg (vector, scales perfectly)

To add PNG fallback for older browsers:
1. Convert favicon.svg to 32x32 PNG
2. Save as static/favicon.png
3. The HTML already references both formats

Tools to convert:
- Online: https://convertio.co/svg-png/
- Or use any image editor (Photoshop, GIMP, etc.)

The SVG favicon works in all modern browsers!
